# Yellow Foods MongoDB
A MongoDB-backed REST API in JavaScript and Express.js:
```HTTP
GET api/foods/5f77ba2bfe614c1ac4e63a3d
```

```JavaScript
{
    "_id": 5f77ba2bfe614c1ac4e63a3d,
    "name": "banana"
}
```

## Prerequisites:
- Node.js 12.18.4
- MongoDB 4.4.1

## Build and run:
### Node.js / open:
- `npm install`
- `npm start`
