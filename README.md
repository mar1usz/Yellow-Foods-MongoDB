# Yellow Foods MongoDB
A MongoDB-backed REST API in JavaScript (Node.js) and Express.js:
```HTTP
GET api/foods/5f77ba2bfe614c1ac4e63a3d
```

```JavaScript
{
    "_id": "5f77ba2bfe614c1ac4e63a3d",
    "name": "banana"
}
```

## Prerequisites:
- Node.js 15.6.0
- MongoDB 4.4.3
- MongoDB Database Tools 100.2.1

## Build and run:
### nodemon / open:
- `npm update`
- `npm install`
- `npm start`
